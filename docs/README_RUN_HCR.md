## Running the HCR system

To start the system:

```
  source ~/.cshrc
  start_all
```

To monitor the system:

```

  # see running processes
  ppm

  # check all processes are running
  pcheck

  # check data flow
  pdm

```

NOTE: TerrainHtServer takes a while to start up and register with procmap.

To stop the system:

```
  stop_all
```
